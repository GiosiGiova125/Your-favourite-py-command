📖 What is this?
This is a simple random command generator. I built it from scratch to learn the fundamentals of programming: **lists**, **variables**, and the **random module**. It's a small step for a developer, but a huge leap for me!

🛠️ How it works
The program greets the user and then **picks a random string from a predefined list of commands.** It’s a great example of how Python handles random logic and user output.
